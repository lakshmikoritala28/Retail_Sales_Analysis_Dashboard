# 📊 Sales Data Analysis & Dashboard

## 📌 Project Overview
This project focuses on analyzing retail sales data to extract meaningful business insights such as revenue trends, top-performing products, and regional performance. An interactive dashboard is built using Python (Pandas, Plotly) and/or Power BI for visualization.

---

## 🎯 Project Objective
- Clean and prepare raw sales data for analysis  
- Perform exploratory data analysis (EDA)  
- Identify key business insights such as top products, regions, and trends  
- Build an interactive dashboard for data visualization  

---

## 🛠️ Tools & Technologies
- Python 🐍  
- Pandas 📊  
- Plotly / Dash 📈  
- Power BI 📊  
- Jupyter Notebook  

---

## 📂 Dataset Description
The dataset contains retail sales transaction details including:

- Order ID  
- Order Date  
- Product Name  
- Category  
- Region  
- Sales  
- Quantity  

---

## 🧹 Data Cleaning Process
- Removed duplicate records  
- Handled missing/null values  
- Converted date column into proper datetime format  
- Ensured data consistency and accuracy  

---

## 📊 Analysis Performed
- Monthly revenue trends analysis  
- Category-wise sales performance  
- Top 10 products by revenue  
- Top 10 regions by revenue  
- Year-over-Year (YoY) growth analysis  

---

## 📈 Dashboard Features
- KPI Cards:
  - Total Sales  
  - Total Orders  
  - Total Quantity  

- Visualizations:
  - Sales by Product (Bar Chart)  
  - Sales by Region (Bar Chart)  
  - Monthly Sales Trend (Line Chart)  
  - Category-wise Sales  

- Filters:
  - Year  
  - Region  
  - Category
